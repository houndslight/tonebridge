# tonebridge.

**Zero-effort DAW-to-stream loopback for guitarists & singers.**

Play guitar, sing, run all your effects in Reaper (or any DAW), hear yourself with low latency, and have the fully processed audio automatically show up in OBS, Discord, YouTube, and Twitch — no fiddling every time you sit down.

Built for my Focusrite Scarlett Solo 1st Gen (and works with most interfaces).

## Why It Exists

Most musicians fight with ASIO vs WDM, Direct Monitoring doubling, and OBS not seeing their effected sound. LoopForge fixes that with one small app that sets up the virtual cable routing and guides you through the rest.

## Quick Start

1. Download the latest release (single .exe)
2. Run as administrator the first time
3. Follow the on-screen setup
4. Open your DAW + OBS/Discord → done

[Download Latest Release](https://github.com/houndslight/tonebridge/releases)

## Features

- Auto-detects and installs VB-Audio Virtual Cable if missing
- Handles Scarlett Solo 1st Gen Direct Monitor warning
- One-click test signal to verify routing
- Persistent configuration (survives reboots)
- Works with OBS, Discord, Streamlabs, etc.
